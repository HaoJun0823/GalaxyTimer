此项目使用教程：注册Github → 复刻此项目到自己的仓库 → 仓库的设置 → 机密与变量 → 新建一个名为HASHIDS_SALT的仓库机密 → 该机密的值填入你自己创建的16位字符 → 保存 → 操作 → 运行工作流程 → 点击最新的工作流程 → 下载打包完成的SaveGalaxyTimer → 解压缩 → 将文件放到你的星河计时器原本的文件夹 → 右键以管理员运行。
